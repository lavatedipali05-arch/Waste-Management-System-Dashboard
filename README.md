# Waste-Management-System-Dashboard
♻️ Waste Management System Dashboard

📊 Overview

This project is an interactive Waste Management Analytics Dashboard built using Streamlit.
It analyzes global plastic waste data and provides meaningful insights through visualizations and predictive modeling.

---

🚀 Features

- 📈 Data Visualization (Bar Chart, Pie Chart, Line Chart)
- 🌍 Top Waste Producing Countries Analysis
- 🔍 Interactive Filters
- 🤖 Machine Learning (LSTM) for Future Waste Prediction
- 📊 Key Insights & Trends

---

🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- NumPy

---

📂 Dataset

- Plastic Waste Around the World Dataset (CSV)
